# 📝 JustNote – Notes Application

A simple and responsive Notes Application built using HTML, CSS, and JavaScript that allows users to create, edit, and delete notes efficiently.

---

## 🚀 Features
- Create, edit, and delete notes
- Data stored using Local Storage
- Responsive UI design
- Simple and user-friendly interface

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure
JustNote/
│── index.html
│── style.css
│── script.js

---

## 🎯 Purpose
This project was built to practice JavaScript concepts like DOM manipulation, local storage, and event handling while improving frontend development skills.
